# Tone Crafter

Tone Crafter 是一个使用 Flutter 构建的 iOS 风格音乐创作应用，提供以下核心功能：

- 🎹 **音色键盘**：支持钢琴、电子琴与自定义音色三种模式，可直接录制并保存键盘音色，同时提供拟真的琴键演奏体验。内置钢琴/电子琴音色由算法实时合成，无需额外音频资源。
- 🥁 **3×3 打击垫**：九个鼓垫可分别录制独立音色，点击即可触发播放，长按即可开始或结束录音。
- 🎼 **音序器**：16 步音序器可编排键盘与鼓垫音色，支持调整 BPM、设置音符音高并实时回放。

## 运行方式

1. 安装 Flutter（建议 3.10 及以上版本）。
2. 在项目根目录执行依赖安装：

   ```bash
   flutter pub get
   ```

3. 运行到模拟器或真机：

   ```bash
   flutter run
   ```

首次运行时请授予麦克风访问权限，以便录制自定义音色。

> 提示：项目不再包含二进制音频文件，如需替换合成音色，可在应用内录制或将自定义样本放置到模拟器/真机的应用沙盒中。
